俄罗斯方块

基于原生 JavaScript + Canvas 实现的经典俄罗斯方块游戏，完整实现了方块七种形态的旋转、下落、碰撞检测、行消除与计分系统。项目采用事件驱动架构，支持键盘与触屏双模式操控，并提供多档难度递进与本地排行榜功能，是前端游戏开发的一次完整实践。

项目背景

俄罗斯方块（Tetris）作为电子游戏史上最具影响力的经典游戏之一，其核心玩法涉及空间思维、实时决策与算法逻辑。本项目以"用前端技术还原经典"为目标，从零开始实现一套完整的游戏引擎。

项目旨在通过游戏开发实践深入理解 JavaScript 事件循环、Canvas 渲染机制、碰撞检测算法以及游戏循环设计模式，为后续更复杂的游戏或交互应用开发积累经验。

核心功能

完整体验经典玩法

S

方块系统

实现七种标准方块（I/O/T/S/Z/J/L）的生成、旋转与预览，旋转采用 Super Rotation System 算法。

C

碰撞检测

基于网格矩阵的实时碰撞检测，支持边界碰撞、方块间碰撞以及旋转合法性校验。

R

行消除系统

支持单行、双行、三行和四行消除判定，消除动画过渡自然，整体计分随消除行数呈指数增长。

L

难度与排行榜

随消除行数自动递增下落速度，提供本地 localStorage 持久化排行榜，记录最佳成绩。

技术实现

游戏主循环基于 requestAnimationFrame 实现，保证不同刷新率设备上的一致体验。方块使用二维矩阵表示旋转状态，通过预计算旋转表避免实时计算开销。碰撞检测采用"试移再判"策略：先尝试移动/旋转，再验证新位置是否合法。

  • 使用 Canvas 2D API 完成全部游戏画面渲染，无需任何第三方库
  • 键盘 WASD/方向键操控 + 触屏手势滑动的双模式输入系统
  • "幽灵方块"预览功能，显示当前方块预计落点位置
  • 下一个方块预览区域，帮助玩家提前规划放置策略
  • 暂停/继续、重新开始等游戏状态管理

设计亮点

  • 模块化架构设计：将渲染、逻辑、输入、音效拆分为独立模块
  • 分数倍增机制：连续消除行数越多，单次得分指数增长
  • 响应式 Canvas 布局，适配移动端与桌面端不同屏幕尺寸
  • 触屏手势识别：支持滑动方向控制与点击旋转
