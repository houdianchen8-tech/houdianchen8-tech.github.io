停车场管理系统

基于 Spring Boot + MySQL 实现的停车场智能管理平台，涵盖车位管理、车辆出入登记、计费结算、会员管理以及报表统计等核心模块。系统采用 RESTful 接口设计与 JWT 权限控制，支持管理员、收费员、会员三种角色分级，通过算法优化车位分配与费用结算流程。

Spring Boot MySQL RESTful API JWT MyBatis Maven

核心模块

涵盖停车场运营全流程

P

车位管理

实时展示车位占用状态，支持按区域、楼层的结构化车位管理以及空闲/占用/预约状态切换。

C

车辆出入

车辆入场自动记录与出场结算处理，支持车牌号快速检索与历史停车记录查询。

$

计费结算

支持按时计费、按次计费、日封顶等多种计费策略，自动计算停车费用并生成结算单。

M

会员管理

会员注册、充值、月卡续费管理，会员车辆自动识别与优惠费率自动应用。
